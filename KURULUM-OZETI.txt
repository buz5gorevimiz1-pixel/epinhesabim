⚙️  TEKNIK DETAYLAR:

Backend (server.js)
───────────────────────────────────────────────────────────────────────────────
Endpoints:
  POST   /api/auth/register   → Yeni kullanıcı kaydı
  POST   /api/auth/login      → Giriş (token al)
  GET    /                    → Ana sayfa (index.html)
  GET    /giris-yap           → Login sayfası
  GET    /admin               → Login sayfası (alternatif)
  GET    /uyelik              → Register sayfası

Database: MongoDB (localhost:27017/itemci)
  Collection: users
    • fullName (string)
    • email (string, unique)
    • phone (string)
    • username (string, unique)
    • passwordHash (bcrypt)

Auth: JWT Token (30 gün geçerli)
  Şifre: Bcryptjs (10 salt rounds)

Frontend (login.html, register.html)
───────────────────────────────────────────────────────────────────────────────
Login Flow:
  1. Form submit → JavaScript intercept
  2. /api/auth/login POST request
  3. Response.ok? Evet → localStorage token kaydı
  4. Redirect → / (index.html)
  5. Hayır → Error message Swal

❌ SORUN GIDERME:

Sorun: "MongoDB Error: connect ECONNREFUSED 127.0.0.1:27017"
Çözüm: MongoDB sunucusu çalışmıyor
   → CHECK-MONGODB.cmd ile kontrol et
   → mongod komutunu çalıştır

Sorun: "HTTP 405 Method Not Allowed"
Çözüm: Form POST request gönderiyor ama handler yok
   → Çözüldü! login.html ve register.html AJAX kullanıyor

Sorun: "Cannot find module 'express'"
Çözüm: Packages yüklü değil
   → INSTALL-DEPENDENCIES.cmd çalıştır

Sorun: "Giriş yapılıyor ama boş sayfa yükleniyor"
Çözüm: index.html frontend kodunda sorun olabilir
   → Tarayıcı console'unda F12 → hatayı bak

Sorun: "CORS hatası"
Çözüm: Backend'de cors() middleware aktif
   → Olmaması gerekli, server.js kontrol et
