Authentication
  • Strategy: JWT (JSON Web Token)
  • Expiry: 30 days
  • Secret: process.env.JWT_SECRET || 'itemci_secret_123'
  • Hash: bcryptjs (10 salt rounds)

Database
  • Provider: MongoDB
  • Connection: mongodb://127.0.0.1:27017/itemci
  • ORM: Mongoose 9.6.2
  • User Model: User.js

API Endpoints
  • POST /api/auth/register  → { full_name, email, phone, username, password }
  • POST /api/auth/login     → { identifier, password }
  • GET  /                   → index.html
  • GET  /giris-yap          → login.html
  • GET  /admin              → login.html
  • GET  /uyelik             → register.html

Frontend Libraries
  • Fetch API (modern XMLHttpRequest replacement)
  • SweetAlert2 (alert/modal UI)
  • FontAwesome (icons)
  • localStorage (browser storage)

Browser APIs Used
  • localStorage.setItem() - Token kaydı
  • localStorage.getItem() - Token okuması
  • fetch() - HTTP requests
  • addEventListener() - Event handling
  • querySelector() - DOM selection
